## Api status codes
### HTTP GET
 - 200 (OK)
 - 404 (NOT FOUND)
 - 400 (BAD REQUEST)
 
### HTTP POST
 - 201 (CREATEED) - Created with url in response
 - 200 (OK)
 - 204 (NO Content)
 
### HTTP PUT
 - 201 (CREATEED) - Created
 - 200 (OK) - Updated
 - 204 (NO Content)
 
### HTTP DELETE
 - 200 (OK) - Deleted
 - 202 (ACCEPTED) - Queued
 - 204 (NO CONTENT)
